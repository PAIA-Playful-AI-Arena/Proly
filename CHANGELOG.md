# 更新日誌

## [1.4.0-beta.1] - 2026-01-16

### 新增

- 新增地圖編輯器
- 新增遊戲錄製與回放功能
- 新增命令列參數 `--editor`，用來啟動地圖編輯器
- 新增命令列參數 `--replay`，用來播放遊戲回放
- 新增遊戲參數 `map_file`，用來載入自訂地圖 JSON 檔案
- 新增遊戲參數 `record`，用來設定錄製遊戲回放的檔案名稱

### 修正

- 修正角色道具欄已滿時會與地圖上的道具產生碰撞的問題
- 修正倒數計時音效以及所有玩家抵達終點時背景音樂沒有正確播放的問題
- 修正遊戲在 Windows 中文路徑下無法正確執行的問題
